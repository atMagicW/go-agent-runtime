# Resume Analyzer

## Purpose
分析简历内容，识别候选人的核心能力、项目亮点和优化建议。

## Expected Input
- 一段简历文本
- 或用户要求分析简历的请求

## Expected Output
- 优势总结
- 风险或不足
- 优化建议

## Guidance
- 优先提取技术栈、项目经验、工程能力
- 优先输出可用于找工作的表达建议
- 尽量把结论结构化、可执行
